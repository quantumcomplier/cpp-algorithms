<h1>Bubble Sort Demonstration</h1>

<p>
This program provides a simple example of the bubble sort algorithm using a small
set of integers. The goal is to show the basic mechanics of the algorithm without
adding unnecessary complexity. The code walks through the list, compares adjacent
values, and swaps them when needed. After each pass, the largest value settles at
the end of the list, and the range of comparison shrinks.
</p>

<h2>What the Program Does</h2>

<ul>
  <li>Creates a small vector of integers</li>
  <li>Prints the original unsorted list</li>
  <li>Runs bubble sort on the data</li>
  <li>Prints the sorted list</li>
</ul>

<p>
The implementation keeps the logic straightforward so the algorithm is easy to follow.
It uses a flag to track whether any swaps occurred during a pass, and the loop ends
once the list is fully sorted.
</p>
