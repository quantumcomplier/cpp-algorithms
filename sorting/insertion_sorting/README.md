<h1>Insertion Sort Demonstration</h1>

<p>
This program shows a simple implementation of the insertion sort algorithm using a small
set of integers. The idea behind insertion sort is straightforward: move through the list
one element at a time and place each value into its correct position relative to the
elements that came before it. The result is a sorted list built gradually from left to right.
</p>

<h2>What the Program Does</h2>

<ul>
  <li>Creates a small vector of integers</li>
  <li>Prints the original unsorted list</li>
  <li>Runs insertion sort on the data</li>
  <li>Prints the sorted list</li>
</ul>

<p>
The implementation keeps the logic clear and easy to follow. Each pass selects a value,
shifts larger elements to the right, and inserts the value into its correct position.
This makes insertion sort a good example for understanding how sorting algorithms work
step by step.
</p>
