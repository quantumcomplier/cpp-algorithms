<h1>Selection Sort Demonstration</h1>

<p>
This program shows a straightforward implementation of the selection sort algorithm.
The idea behind selection sort is simple: move through the list, find the smallest value
in the remaining unsorted portion, and place it into its correct position. The process
repeats until the entire list is sorted.
</p>

<h2>What the Program Does</h2>

<ul>
  <li>Creates a small vector of integers</li>
  <li>Prints the original unsorted list</li>
  <li>Runs selection sort on the data</li>
  <li>Prints the sorted list</li>
</ul>

<p>
The implementation keeps the logic easy to follow. Each pass identifies the smallest
value in the unsorted section and swaps it into place. This makes selection sort a
good example for understanding how sorting algorithms operate step by step.
</p>
