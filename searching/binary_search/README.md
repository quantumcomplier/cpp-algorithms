<h1>Binary Search Demonstration</h1>

<p>
This program provides a simple example of the binary search algorithm. Unlike linear
search, binary search works only on sorted data and uses a divide‑and‑conquer approach
to locate a target value. By repeatedly checking the middle element and narrowing the
search range, the algorithm finds values efficiently.
</p>

<h2>What the Program Does</h2>

<ul>
  <li>Creates a sorted vector of integers</li>
  <li>Prints the list of values</li>
  <li>Prompts the user for a number to search for</li>
  <li>Runs binary search on the data</li>
  <li>Displays whether the value was found and its index</li>
</ul>

<p>
The implementation keeps the logic clear and easy to follow. The program uses left,
right, and mid indices to narrow the search range until the target is found or the
range is exhausted. This makes binary search a good example of how algorithmic
efficiency improves when data is structured.
</p>

<h2>Concepts Demonstrated</h2>

<ul>
  <li>Divide‑and‑conquer search logic</li>

  <li>Midpoint calculation</li>
  <li>Adjusting search boundaries</li>
