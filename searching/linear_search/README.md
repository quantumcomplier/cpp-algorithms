<h1>Linear Search Demonstration</h1>

<p>
This program provides a simple example of the linear search algorithm. The idea behind
linear search is straightforward: move through the list one element at a time and check
each value until the target is found. If the value appears in the list, the algorithm
returns its index. If not, it reports that the value is missing.
</p>

<h2>What the Program Does</h2>

<ul>
  <li>Creates a small vector of integers</li>
  <li>Prints the list of values</li>
  <li>Prompts the user for a number to search for</li>
  <li>Runs linear search on the data</li>
  <li>Displays whether the value was found and its index</li>
</ul>

<p>
The implementation keeps the logic clear and easy to follow. Each element is checked
in order, which makes linear search a good starting point for understanding how search
algorithms work before moving on to more efficient methods.
</p>
